# Backend Java (Spring Boot)

This directory is a placeholder for the Java Spring Boot service.

## Setup

1. Copy your Spring Boot project into this directory
2. Ensure your `pom.xml` or `build.gradle` is at `backend-java/` root
3. Configure environment variables in the root `.env` file

## Deployment

This service will be deployed as a separate container on Coolify.

## Structure (expected)

```
backend-java/
├── src/
│   └── main/
│       ├── java/
│       └── resources/
├── pom.xml (or build.gradle)
└── Dockerfile
```
